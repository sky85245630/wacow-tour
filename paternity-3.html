親子同遊-墾丁

第一天

台北/台中/高雄 → 澎湖 → 夜逛馬公西門町
【前往澎湖】
整裝行囊，飛往充滿活力的澎湖。你可從北中南各地，搭乘遠東航空 大飛機直飛。我們將貼心為你準備出發機上餐盒。 午餐：Ｘ
晚餐：西點餐盒
住宿：自選等級住宿

第二天

潮間帶踏浪尋寶 → 海上生態巡航 → 澎澎灘勁爆 水上活動 → 花菜干人文餐廳
【潮間帶】
澎湖潮間帶落差大為踏浪天堂，穿著膠鞋帶一同來去體驗潮間帶的海底生 態，抓螃蟹、玩海星、摸海膽、品嚐新鮮的蚵仔、體驗親手餵魚的樂趣。
【澎澎灘】
勁爆水上活動，水上機車、香蕉船、鴛鴦飛船、水上拖曳圈等多項不限次 數主題式玩法，讓您夏季戲水“FUN”個夠，Hi到最高點！ （不合適水上活動者，可將水上活動行程改ㄨ為趣味海釣！）
【地質景觀】
搭乘遊艇前往，鳥嶼~漁村景麗、丁香魚的故鄉。員貝後山~文房四寶、狀 元穴、崢嶸的玄武岩奇觀。無人島巡航，雞善嶼~燕鷗覓食區賞鳥者天堂、 錠鉤嶼~澎湖小桂林，保證讓你不斷的按下相機快門。
午餐：東海活動餐
晚餐：花菜干人文懷舊餐廳
住宿：自選等級住宿


第三天

海底總動員浮潛體驗 → 白灣海景餐廳 → 沙灘親子 趣味運動會 → 無限自助碳烤BBQ
【浮潛體驗】
專業的配備+教練細⼼的教學，不會游泳與第⼀次浮潛都不用擔心喔！ 備有最完善的裝備：潛水⾐、潛⽔專用膠鞋、各種size的救生衣、面鏡。 期間可看到⼩丑⿂、海葵、藍雀鯛、熱帶魚、河豚、海參等海洋生物。
【沙灘運動會】
英雄大會師、風起雲湧小隊呼精神大比拼！爸爸媽媽與小朋友共同完成各 種關卡闖關活動。沙灘烏龜翹、天玄地轉、你丟我撿、我是最佳造型師。
午餐：白灣海景餐廳
晚餐：無限自助BBQ碳烤
住宿：自選等級住宿


第四天

晨喚 → 全日機車趴趴GO自由行
【機車自由行】
機車是澎湖旅遊最方便的交通工具。 讓我們騎乘機車徜徉在夏日的氣息中。
午餐：自行品嚐在地美味
晚餐：自行嚐鮮特色小吃
住宿：自選等級住宿


第五天

晨喚 → 馬公機場 → 台北/台中/高雄
【期待再相會】
收拾行囊，揮別澎湖。期待下一次我們一起的旅行。 專程為你準備澎湖當地特色早點『二信飯糰』。
特別說明：行程順序將視潮汐及天候狀況作適當之順序調整。
