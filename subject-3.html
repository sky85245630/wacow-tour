第一天

桃園機場集合出發 → 泰國曼谷機場 → 綠山國 家公園 → JUNGLE FLIGHT → 金三角風情 → 芭達雅美軍俱部洋人街
【綠山國家公園】
充滿濃濃泰式風情的綠山國家公園，是東南亞最大的開放式動物 園。搭乘遊園公車，可以發現可愛的鹿群悠閒地從身旁經過，不 怕生地與您接觸！這裡的動物不僅可供近距離觀賞，更可以餵食 互動，除了與動物之間的互動，綠山國家動物園還準備精彩動物 實境秀，讓您欣賞動物的神采模樣，種種難得畫面令人嘖嘖稱奇。
【JUNGLE FLIGHT】
起源於2009年是清邁近兩年來新興的戶外活動，以前所未有的 方式體驗泰國叢林之美，讓來到清邁的遊客能感受到全然不同 的泰國原始風情，深入其境感動與驚奇的程度是難以溢於言表 體驗活動：飛越鱷魚潭、滑翔樂歡樂7合1、雙人騎象、趣味釣 鯊魚、JUNGLE FLIGHT、ATV四輪傳動、猴子趣味表演.....
【金三角風情】
金三角異域風情博物館是著名的泰國旅遊景點之一，收藏了一段 60多年前國共內戰的血淚史。於1990年朱延平執導「異域」片 中主角的照片及史料，完整保存「異域博物館」中。在這裡體驗 泰北村落、異域博物館的歷史故事、泰國毒王坤沙記錄影片、 再讓你熱帶水果吃到飽...
早餐：機上套餐
午餐：港式飲茶風味餐
晚餐：水果養生餐＋巧克力鍋

第二天

早安芭達雅 → 白沙星月島：航海王爭霸戰、水 上活動十二合一 → 南洋薰香泰式皇家ＳＰＡ → 漫遊皇家花園廣場
【航海王爭霸戰】
前往白沙星月島，安排避開陸客的水上活動海域，享受暢完最創 新的12合1水上活動全日遊：空中飛人托曳傘、刺激翻滾香蕉船 、奔馳水上摩托車、白沙星月島如畫逍遙風情游、與熱帶魚共舞 浮潛樂、勁爆海上溜滑梯、挑戰高空跳海冒險、天才小釣手專業 海釣、海灘星光大道中文卡拉OK歡唱、體驗船上大舕海鮮餐、 特別贈送可樂.啤酒無限暢飲喝到飽、當季水果大快朵頤...
【薰香皇家SPA】
項目包括：芳香精油指壓、皇家天然藥草蒸氣浴、品茗養生茶 ，陪櫬於周圍的優雅環境在高級美容師親切細心專業的服務下享 受泰式精油按摩，及多項設施將可使您充分體驗到泰式SPA的樂 趣及享受，讓你真正的陶醉在SPA的樂趣中，徹底的讓身心得到 完全的解放。
【皇家花園廣場】
座落當地最熱鬧的芭達雅濱海道路上，並擁有寬敞舒適的購物空 間，成為前往芭達雅的旅客必去朝聖的精品商場之一，此外，這 裡除了販買泰國的特色小物，還包括曼谷包、Boots系列商品、 必買的金湯普森Jim Thompson House高級泰絲、包含泰國當紅 品牌Be Yourself也在這裡設有專櫃，五花八門、無所不包，喜愛 掃街買逛的朋友，千萬別錯過最in的購物廣場。
早餐：酒店內
午餐：船上海鮮風味餐
晚餐：泰式風味餐
住宿：HOLIDAY RESORT 或 EASTINY REOSRT 或同級


第三天

日出芭達雅 → 空中展望台眺望芭達雅 → 刺激 高空滑樓挑戰 → 四方水上市場 → 泰式廚藝料 理體驗 → 羅馬競技場歌舞人妖秀
【高空滑樓挑戰】
首先來到Pattaya Park 55樓，在這裡可以俯瞰整個芭達雅的景色 。再來就是重頭戲高空滑降，可選擇三種跳樓方式喔！ 『雙手滑跳』1個人雙腳懸空、『雙人同跳』、『高空纜車』，從 56層跳下來的滋味，保證讓您回味無窮，永難忘懷！真的不敢跳 的人，還有第四種：就是直接搭電梯下去…
【四方水上市場】
電影杜拉拉升職記拍攝的場景，結合農村文化藝術元素，融合鄉 村文化木雕風格的複合式水上市場。各種泰式水上建築在迂迴的 河道上，特別安排搭乘『手搖舢舨船』，踅遊穿梭在水上五金、 水果、雜項商家，民俗傳統舞蹈笙歌，木雕藝術敲打聲中。
【泰式廚藝體驗】
廚藝料理體驗基本教學的內容有泰式酸辣海鮮湯、涼拌青木瓜沙 拉。體驗試著用不認識的香料相互組合，完成令人意想不到的美 味料理，感受泰國菜真正的趣味。
【羅馬競技場人妖秀】
仿羅馬神鬼戰士鬥獸場外觀，廳內歐式巴洛克裝潢；亮麗 典雅。耗費巨資的表演秀，常讓中外旅客讚賞不以。
早餐：酒店內
午餐：52樓旋轉餐廳
晚餐：活跳跳海鮮燒烤＋啤酒
住宿：HOLIDAY RESORT 或 EASTINY REOSRT 或同級


第四天

早安芭達雅 → 曼谷時尚精品採購、特產巡禮 → 毒蛇研究中心→ 曼谷四面佛 → 正宗泰式古法 按摩 → 威尼斯碼頭市集ASIA TIQUE
【特產採購巡禮】
皮件展示中心、皇家珠寶中心、毒蛇研究中心。泰國獨有魚種～ 珍珠魚，製作成精美時尚的皮夾、皮包，可以刷洗堅實耐用。 泰國是世界紅寶石的主要加工區，泰國寶石都受到世界珠寶玉石 鑒定協會認證。毒蛇研究中心觀賞人蛇大戰，並可瞭解到毒蛇的 功用及遇蛇急救之方法...
【曼谷四面佛】
四面佛第一個面是求生意、第二面求愛情、第三面求錢財、第四 面求平安。依據泰國當地人的說法，四面佛怎麼拜？要以順時針 的方向由第一面進行參拜，而且要許的願望對每一面都說一次， 而且在許願的當下就要把還願的方式稟告神明，聽說還願的方法 越稀少，願望越容易實現。
【泰國古式按摩】
按摩在泰國古代講求的其實是「醫學」效果，將近有4千多年歷 史，是泰國相當重要的人文資產，古代泰國皇族利用泰式按摩當 做強身健體的一種方式。
【ASIA TIQUE】
威尼斯碼頭市集是目前泰國最火紅的市集，好比台灣的夜市，但 其規模及好逛的程度，絕對超過台灣的任何一個夜市，我們一起 來體驗一下曼谷的夜生活吧！
早餐：酒店內
午餐：采華軒自助餐
晚餐：自理 (威尼斯碼頭市集)
住宿：THE TIVOLI HOTEL 或 VIC 3 或同級


第五天

日出曼谷→ 五世皇柚木皇宮 → 曼谷中央購物 中心 → 曼谷機場 → 桃園機場 → 期待再相會
【五世皇柚木皇宮】
拉瑪五世接受西方教育，繼位之後不負父望，不但廢除奴隸 制度，在位期間對司法和財政也進行大力改革，對泰國歷史 演變影響極其重大，柚木行宮正是紀念這位偉人的博物館。 建造於西元1900年，是採用金柚木所建造的三層建築，所以 又稱為金柚木宮。行宮外是一大片綠色草坪和花圃，建築風 格以泰國傳統為基礎，但明顯受到歐洲建築設計的影響。柚 木行宮並且使用全泰國品質最優良的金柚木建造，建築規模 堪稱世界最大，但卻沒有一根鐵釘子在上面。
【中央購物中心】
曼谷Central World設計新潮時尚，聘請美、德建築師創新使 用長達300公尺長的「玻璃式帷幕」，讓整棟建築有如大型透 明展示櫥窗，頗有藝術風格；入夜後打亮LED螢幕，更加顯眼 閃亮。泰國Central World Plaza主要是由3個百貨公司組成的 複合式商圈，分別是主體的Central World、以曼谷當地品牌 服飾為主的Zen，以及日系商品為主的伊勢丹。許多前往曼谷 的購物達人，來到曼谷都必買必逛的Central World。
早餐：酒店內
午餐：泰式風味餐
晚餐：機上套餐
